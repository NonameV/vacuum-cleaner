How Do I Love Thee?

Having two rat terriers who shed like crazy and a strange crop of tiny black bugs made me obsessed about finally getting a cordless stick vacuum. This one by Samsung is fantastic. It’s very light and easy to clean. I very much appreciate the fact that unlike some other cordless vacuums the battery can be charged separately from the main unit so I did not need to install the mounting system. The charging base is convertible so it can function as either a mounting charger and storage for the vacuum or can be easily reconfigured to be a table top charger for just the battery and the rest of vacuum can be stored in a closet. This system makes it possible to get an extra battery to keep on hand for extra cleaning time if needed. I have found the charge time perfectly adequate for my needs so far. My house is so much cleaner. So easy to grab it and pick up bugs and dog hair as needed!
